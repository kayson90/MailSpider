High Level业务需求

EAS Ops在设计未来的价格审批流程，其中需要一个工具记录价格审批的历史邮件记录，并且自动汇总、整理成Excel表格备查。

 

建议的实现方式

申请一个公司的Function Outlook Account （比如EASPriceApproval@lenovo.com），每次价格审批的发起、审批相关邮件都要抄送这个“邮箱”，“工具”定期自动读取“邮箱”的历史邮件，读取邮件内容中固定格式的表格，把表格中的信息提取并写入Excel文件，作为历史数据存档备查。

 

具体需求

1． “工具”定期读取“邮箱”中的“未读”邮件

a)      Case a:新建待审批商机。 - “工具”读取邮件中的表格（模板见下）,发现发件人是表格中的提交人，并且“商机编号”未存在历史记录中，则读取表格内容，在Excel中新建一条记录，并记录发邮件时间。

b)      Case b:审批人审批记录。 -  发现发件人是表格中的审批人（各个级别），并且“商机编号”已存在历史记录中，并且有审批结果，则读取表格内容，在Excel中生成一条记录，并记录发邮件时间。

c)      Case c: 商机审批完成记录。 - 发件人是表格中的审批人（最后一个级别），并且“商机编号”已存在历史记录中，并且有审批结果，并且所有级别都审批完成，则读取表格内容，在Excel中生成一条记录，标识商机审批完成，并记录发邮件时间。

如果同一个审批人多次发出邮件，并且带有审批结果，则生成多条记录。

2． “工具”把处理过的邮件设置成“已读”。

3． Excel中需要包括审批模板中的所有数据项，并且附加上“审批时间”，“是否完成审批”标志位。

4． Standalone 和Soft Bundled的商机审批模板不同，需要分别记录。
